# hubot-zoi

今日も一日がんばるぞい！

## Install

```bash
npm install --save hubot-zoi
vim external-scripts.json
# Edit => ["hubot-zoi"]
```

### Configure

Set the `HUBOT_ZOI_SOURCE_URL` environment variable to the images source(YAML is recommended).

`hubot-zoi` gets the image url randomly from src.
